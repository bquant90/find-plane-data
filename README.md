# find-plane-data
A C program that reads in plane data via text and allows for manipulation of plane information, based on user input.

# Project Origins and Evolution (Disclaimer)
This project originated from a university course assignment but with significant refactoring and enhancement, such as:
- Modular design principles
- Enhanced user experience with clear prompts and feedback
- Comprehensive input validation and error handling
- Option to edit and display new plane information

# Technical Details
- Time Complexity: O(planes) for each plane record in the text file
- Space Complexity: O(planes) for each plane record in the text file
- Language: C99 or later
- Key Features: Menu-based user interaction, plane data editing, file input/output
